# Bank

Not a real bank app. Python based.

## Stack is
* SQLITE
* sqlite3 (DAO)
* fastapi (API)
* front
  * flask (web UI)
  * Python script

---

## Database
* run db/bank.sql to setup

## Build
* `pip install -r requirements.txt`

## Run API
* `uvicorn main:app --reload` in root directory

## Run UI
* `flask run` in ui directory

## Python frontend
* run desktop/main.py
